
3. Build the project using Maven:

    ```bash
    ./mvnw clean package
    ```

4. Run the application:

    ```bash
    java -jar target/yachun-0.0.1-SNAPSHOT.jar.original
    ```

5. The application will start running at `http://localhost:8080`.
